== PWM und LED-Dimmer

image::bilder/35_pwm.png[align=center]

=== Das lernst du

In diesem Kapitel lernst du:

* was PWM (Pulsweitenmodulation) ist
* wie PWM funktioniert
* was ein Tastverhältnis ist
* wie eine LED stufenlos gedimmt werden kann
* wie PWM als analoger Ausgang verwendet wird
* wie ein Potentiometer die Helligkeit einer LED steuern kann

Am Ende des Kapitels kannst du die Helligkeit einer LED stufenlos verändern und eigene PWM-Anwendungen entwickeln.

=== Warum PWM?

Im letzten Kapitel konnten wir analoge Werte messen.

Mit einem Potentiometer haben wir beispielsweise Werte zwischen:

[source]
----
0
und
1023
----

eingelesen.

Nun möchten wir diese Werte verwenden, um die Helligkeit einer LED zu steuern.

Das klingt zunächst einfach.

Ein Mikrocontroller-Ausgang kennt jedoch normalerweise nur zwei Zustände:

[source]
----
EIN
AUS
----

Wie lässt sich daraus eine halbe Helligkeit erzeugen?

Die Lösung heißt:

[source]
----
PWM
----

=== Was bedeutet PWM?

PWM steht für:

[source]
----
Pulse Width Modulation
----

auf Deutsch:

[source]
----
Pulsweitenmodulation
----

Dabei wird ein Ausgang sehr schnell ein- und ausgeschaltet.

=== Ein Vergleich

Stell dir eine Lampe vor.

Sie ist immer nur:

[source]
----
EIN
oder
AUS
----

Wird sie jedoch sehr schnell ein- und ausgeschaltet, nimmt unser Auge nur noch eine mittlere Helligkeit wahr.

=== Das Tastverhältnis

Entscheidend ist das Verhältnis zwischen:

* Einschaltzeit
* Ausschaltzeit

Dieses Verhältnis nennt man:

[source]
----
Tastverhältnis
----

oder englisch:

[source]
----
Duty Cycle
----

=== Beispiele

100 % Tastverhältnis:

[source]
----
1111111111111111
----

Die LED ist während der gesamten Periode eingeschaltet.

50 % Tastverhältnis:

[source]
----
1111000011110000
----

Die LED ist etwa gleich lange ein- und ausgeschaltet.

25 % Tastverhältnis:

[source]
----
1100000011000000
----

Die LED ist nur während eines Viertels der Zeit eingeschaltet.

Dabei bedeutet:

[source]
----
1 = Ausgang eingeschaltet
0 = Ausgang ausgeschaltet
----

=== Warum sehen wir kein Flackern?

PWM arbeitet normalerweise mit hohen Frequenzen.

Typische Werte sind:

[source]
----
100 Hz
500 Hz
1000 Hz
----

Unser Auge kann diese schnellen Änderungen nicht mehr erkennen.

Wir sehen lediglich die resultierende Helligkeit.

=== PWM beim PicoMite

Der PicoMite besitzt mehrere PWM-Kanäle.

Ein PWM-Signal wird mit dem Befehl:

[source,basic]
----
PWM Kanal,Frequenz,Tastverhaeltnis
----

erzeugt.

Beispiel:

[source,basic]
----
PWM 0,1000,50
----

Das bedeutet:

[cols="1,3"]
|===
|Parameter |Bedeutung

|0
|PWM-Kanal

|1000
|1000 Hz Frequenz

|50
|50 % Tastverhältnis
|===

=== PWM-Ausgänge konfigurieren

Bevor ein GPIO-Pin verwendet werden kann, muss er einem PWM-Kanal zugeordnet werden.

Beispiel:

[source,basic]
----
SETPIN GP0,PWM0A
----

Nun wird das PWM-Signal des Kanals 0 auf GP0 ausgegeben.

=== Erste PWM-Ausgabe

[source,basic]
----
SETPIN GP0,PWM0A

PWM 0,1000,50
----

Die LED sollte nun etwa halb so hell leuchten.

=== Verschiedene Helligkeiten

25 %:

[source,basic]
----
PWM 0,1000,25
----

50 %:

[source,basic]
----
PWM 0,1000,50
----

75 %:

[source,basic]
----
PWM 0,1000,75
----

100 %:

[source,basic]
----
PWM 0,1000,100
----

=== PWM stoppen

[source,basic]
----
PWM 0,OFF
----

Der PWM-Kanal wird abgeschaltet.

=== Der Colour Maximite 2

Auch der Colour Maximite 2 unterstützt PWM.

Dort werden PWM-Controller verwendet.

Beispiel:

[source,basic]
----
PWM 1,1000,50
----

Dadurch wird auf dem ersten PWM-Ausgang ein Signal mit:

* 1000 Hz
* 50 % Tastverhältnis

erzeugt.

Die Grundidee ist auf beiden Plattformen identisch.

=== Eine LED langsam heller werden lassen

Nun erzeugen wir einen einfachen Aufblendeffekt.

[source,basic]
----
SETPIN GP0,PWM0A

FOR HELLIGKEIT=0 TO 100

  PWM 0,1000,HELLIGKEIT

  PAUSE 20

NEXT HELLIGKEIT
----

Die LED wird langsam heller.

=== Eine LED langsam dunkler werden lassen

[source,basic]
----
SETPIN GP0,PWM0A

FOR HELLIGKEIT=100 TO 0 STEP -1

  PWM 0,1000,HELLIGKEIT

  PAUSE 20

NEXT HELLIGKEIT
----

Die LED wird langsam dunkler.

=== Atemlicht

Kombinieren wir beide Effekte.

[source,basic]
----
SETPIN GP0,PWM0A

DO

  FOR H=0 TO 100

    PWM 0,1000,H

    PAUSE 10

  NEXT H

  FOR H=100 TO 0 STEP -1

    PWM 0,1000,H

    PAUSE 10

  NEXT H

LOOP
----

Die LED wirkt nun, als würde sie atmen.

=== PWM als analoger Ausgang

PWM wird häufig verwendet, um analoge Spannungen zu simulieren.

Betrachten wir folgende Werte:

[cols="1,2"]
|===
|PWM |Mittlere Spannung

|0 %
|0,0 V

|50 %
|ca. 1,65 V

|100 %
|ca. 3,3 V
|===

Mit einem Tiefpassfilter kann daraus eine nahezu analoge Spannung erzeugt werden.

=== PWM in der Praxis

PWM wird verwendet für:

* LED-Dimmer
* Motorsteuerungen
* Lüfterregelungen
* Servoansteuerungen
* Audioausgabe
* Netzteile

Es gehört zu den wichtigsten Techniken moderner Mikrocontroller.

=== Praxisprojekt: Dimmer mit Potentiometer

Nun verbinden wir die Kapitel 34 und 35.

Das Potentiometer liefert den Sollwert.

PWM steuert die Helligkeit.

=== Die Hardware

Benötigt werden:

* Potentiometer an GP26
* LED an GP0
* Vorwiderstand 330 Ω

[source]
----
GP26 -> Potentiometer

GP0 -> LED
----

=== Programm

[source,basic]

SETPIN GP26,AIN
SETPIN GP0,PWM0A

DO

WERT=PIN(GP26)

HELLIGKEIT=WERT*100/1023

PWM 0,1000,HELLIGKEIT

PAUSE 20

LOOP

Nun kann die Helligkeit der LED mit dem Potentiometer eingestellt werden.

=== Erweiterung: Anzeige der Helligkeit

[source,basic]

SETPIN GP26,AIN
SETPIN GP0,PWM0A

DO

WERT=PIN(GP26)

HELLIGKEIT=WERT*100/1023

PWM 0,1000,HELLIGKEIT

CLS

PRINT "Helligkeit:"
PRINT INT(HELLIGKEIT);" %"

PAUSE 100

LOOP

Die aktuelle Helligkeit wird zusätzlich angezeigt.

=== Typische Fehler

Wenn die LED nicht reagiert:

. Ist der richtige PWM-Pin verwendet?
. Wurde SETPIN korrekt aufgerufen?
. Ist die LED richtig angeschlossen?
. Ist ein Vorwiderstand vorhanden?
. Wurde der richtige PWM-Kanal verwendet?
